# Code of Conduct

This project follows a Code of Conduct to foster an open and welcoming environment for all contributors and users.

Be respectful, considerate, and kind. Harassment, discriminatory language, or abusive behavior will not be tolerated.

If you experience or witness unacceptable behavior, please open an issue or contact the maintainers so we can address it promptly.

For a full, canonical version you can adopt or adapt, consider the Contributor Covenant v2.0:
https://www.contributor-covenant.org/version/2/0/code_of_conduct/

Maintainers may update or extend this document as needed; contributors should follow the version included in this repository.
